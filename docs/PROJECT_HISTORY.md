**Receipt auto-crop overhaul: true 4-corner detection + Gemini fallback (2026-06-27).** Replaced the original `detectReceiptBBox` (axis-aligned bounding box) with a new dependency-free `detectReceiptQuad` (`src/utils/receipts/autoCrop.ts`) that returns a true `Quad` (4 tilted corners) to seed the manual draggable crop UI. Designed via FUSION MODE: 5-model panel (Gemini 3.1 Pro, Gemini 3.5 Flash, Claude Opus, DeepSeek V4 Flash, DeepSeek V4 Pro) proposed diffs; judged and synthesized into a three-pass detector:

- **Pass 1 (brightness blob):** `boxBlur3` → Otsu threshold (floor 110) → morphological close (dilate+erode) → largest-component flood-fill → convex hull of boundary points → diagonal-extreme `pickCorners` (TL=min(x+y), BR=max(x+y), TR=max(x−y), BL=min(x−y), tilt-robust to ~45°) → area + min-side validity check → `insetToCentroid(0.015)` → return fractional Quad. Works for white receipt on contrasting background. **False-positive guard added (2026-06-27):** if the bright component's bbox touches all 4 image edges (within 3px), the whole scene is bright (white-on-white) — both Pass 1 return paths are skipped, preventing the "whole image as receipt" failure.
- **Pass 2a (150px Hough — 2026-06-27 addition):** Downsamples `rawLum` by 2×2 box average to ~150px. At half-resolution, dense receipt text blurs into sub-pixel noise (impulse-averaging) while the paper boundary (a step-function edge) survives with preserved amplitude. Runs Sobel + Hough with `minVotes = max(20, sqrt(n)*0.15)`. Primary fix for white-on-white receipts. Chosen over Canny NMS because: on white-on-white the paper-boundary gradient (~10–15 luma) is well below text edges (50–100), so Canny's hysteresis threshold (derived from top-8% of magnitudes) drops the paper edge — downscaling blurs the text without touching the paper step.
- **Pass 2b (300px Hough — original Pass 2):** Full-res Sobel gradient (raw luma, not blurred) → adaptive top-8% edge threshold → `houghLines` (accumulator + NMS, minVotes=max(15,sqrt(n)*0.08)) → `rectangleFromLines` (pair near-H × near-V lines, score=area×votes). Fallback when 2a finds nothing.
